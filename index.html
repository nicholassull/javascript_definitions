Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.
Variable naming conventions
If you can, use 'const'. Otherwise, if a variable will change, use 'let'. At the very least use 'var' and don't just use '=' tp assign a vartiable.
