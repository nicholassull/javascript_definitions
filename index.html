JavaScript Definitions

A study aid

JavaScript (JS)
A scripting language that enables one to create dynamically updates content, animate images, and much more.
Operators
A special character (or characters) that indicates an action to be performed.
Variables
Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.
Variable naming conventions
If you can, use 'const'. Otherwise, if a variable will change, use 'let'. At the very least use 'var' and don't just use '=' tp assign a vartiable.
Functions
A block of code that performs an action and returns a result; optionally takes arguments.
Methods
An action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.
Arguments
Some methods take one or more arguments that provide the method with additional information to help it perform its action.
Parameters
The names or placeholders lited in a function's definition.
Return
What is given as the output of a function.
Chaining methods
To chain methods simply continue attatching methods to the target.
Strings
A data type that represents the exact text of whatever is enclosed in the quotes.
Booleans
A primitive that is either true or false.
Undefined
This means something hasn't been defined in our code yet.

Null
